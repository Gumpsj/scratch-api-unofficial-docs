# User Object

A user object looks like this:

```
{
    "id": /* User ID */
    "username": /* Username */
    "history": {
        "joined": /* Date joined */
        "login": /* Last login date? */
    }
    "profile": {
        "id": /* Profile ID? */
        "avatar": /* Part of a path to avatar? */
        "status": /* What I'm working on */
        "bio": /* About me */
    }
}
```
